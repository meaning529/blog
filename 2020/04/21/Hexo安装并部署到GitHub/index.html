hexo,github
安装blog – sheep
  1. 安装node.js
  2. 进入终端，sudo su
  3. 切换到国内源：npm install -g cnpm –registry=https://registry.npm.taobao.org
  4. 安装hexo：cnpm install -g hexo-cli
  5. 验证：hexo -v
  6. 新建blog目录：mkdir myBlog
  7. 进入sh终端：sudo su
  8. 初始化blog：sudo hexo init
  9. 开始运行：hexo s
  10. 新建文章：hexo n “我的一篇文章”
  11. 清空缓存：hexo clean
  12. 生成：hexo g
