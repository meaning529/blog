Linux端口被占用的解决办法

  1. 查看被占用端口信息
lsof -i:端口号
# lsof -i:4000
COMMAND   PID USER   FD   TYPE   DEVICE SIZE/OFF NODE NAME
nodejs  26993 root   10u  IPv4 37999514      0t0  TCP *:4000 (LISTEN)
  1. 解除端口占用
kill -9 PID
  1. netstat 用于显示tcp、udp等端口和进程的相关信息
netstat -tunlp | grep 端口号
  • -t //显示tcp相关
  • -u //显示udp相关
  • -n //拒绝显示别名，能用数字的全部转化为数字
  • -l //仅列出Listen（监听）的服务状态
  • -p //显示建立相关链接的程序名
